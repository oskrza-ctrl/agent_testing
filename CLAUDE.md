# Instrucciones para Claude Code

- Responde siempre en español.
- Explica los pasos de forma breve y clara.
- Vamos paso por paso.

## Coding Guidelines

- Mantén el código simple, claro y fácil de entender.
- Agrega comentarios breves donde ayuden a entender la intención del código.
- No sobrecompliques la arquitectura si una solución simple funciona.
- No implementes múltiples cambios grandes en una sola iteración.
- Antes de modificar código, explica brevemente el plan.
- Después de modificar código, resume qué cambió y cómo probarlo.
- Mantén main.py como punto de entrada simple.
- Separa responsabilidades en módulos cuando el código empiece a crecer.
- No agregues nuevas dependencias sin explicar por qué son necesarias.
- No cambies el comportamiento existente sin avisar.
