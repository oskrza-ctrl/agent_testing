🔄 PIPELINE — LangGraph StateGraph
pipeline/graph.py · 9 nodos secuenciales · estado compartido PipelineState
🎙️ transcription → 🔍 analysis → 📝 markdown → 🗄️ knowledge_base → ☁️ drive_sync → ✅ tasks → 📅 calendar → 📦 archive → 📁 drive_processed
Estado: mp3_path · mp3_name · mp3_stem · transcript · result · error
Cada nodo verifica state["error"] antes de ejecutar — fallo no bloquea el grafo
